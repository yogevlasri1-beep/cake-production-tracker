# מעקב יצור

אפליקציית PWA לתיעוד ייצור במפעל — עובדת באייפון ובדפדפן, נתונים נשמרים מקומית.

## הרצה מקומית

```bash
cd ~/Projects/cake-production-tracker
ruby -run -e httpd . -p 8765
```

פתח: http://127.0.0.1:8765

## מבנה

| מסך | תפקיד |
|-----|--------|
| בית | גרפים + סיכום חודשי |
| ייצור | רישום מוצרים סופיים |
| תיעוד | תהליכי הכנה (נפרד) |
| מוצרים | קטגוריות ומחירים |
| יעדים | יעדי כמות |
| דוחות | ייצוא Excel |

## גיבוי

הנתונים ב-IndexedDB בדפדפן. לגיבוי — ייצוא דוחות Excel ממסך דוחות.
